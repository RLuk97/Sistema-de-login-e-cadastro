# 🔥 FOGO EM VAGAS - Roadmap de Desenvolvimento

## 🛠️ Stack Técnica Atual
```
Frontend: React.js + CSS3
Backend: Node.js + Express
Database: SQLite (migrar para PostgreSQL)
Auth: JWT + bcrypt
Deploy: Vercel + Railway
```
